🎨 3D卡片轮播组件

这是一个可复用的3D卡片轮播组件，可以从用户提供的播放器代码中提取使用。 支持鼠标拖拽和触摸滑动切换，具有流畅的3D变换效果。

请选择卡片

拖拽或点击切换

📦 如何使用

1. 引入文件

<link rel="stylesheet" href="3d-carousel.css">
<script src="3d-carousel.js"></script>

2. 创建HTML结构

<div id="your-container" class="carousel-stage perspective-1200">
    <div class="carousel-track preserve-3d">
        <!-- 卡片将动态生成 -->
    </div>
</div>

3. 初始化轮播

// 准备数据
const items = [
    {
        title: '章节标题',
        artist: '艺术家',
        cover: '封面URL'
    },
    // ...更多项目
];

// 初始化轮播
const carousel = init3DCarousel('your-container', items, {
    initialIndex: 0,  // 初始选中索引
    onChange: (index, item) => {
        // 切换时的回调
        console.log('切换到:', item.title);
    }
});

4. API方法

  • carousel.next() - 切换到下一首
  • carousel.prev() - 切换到上一首
  • carousel.goTo(index) - 切换到指定索引
  • carousel.getCurrentIndex() - 获取当前索引
  • carousel.getCurrentItem() - 获取当前项目数据
