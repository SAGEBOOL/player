## 集成3D轮播到 he-yi.html 播客播放器

### 任务背景
用户要求：用3D轮播替换现有播放列表UI，保留所有功能（定时关闭、字幕、播放统计等），点击3D卡片流畅丝滑的切换曲目，带动画效果。

### 完成内容

#### 1. 添加3D轮播CSS样式
在 `he-yi.html` 的 `<style>` 标签内添加了：
- `.carousel-stage-3d` - 轮播容器（320×400px，perspective: 1200px）
- `.podcast-3d-card` - 卡片样式（220×300px，圆角、阴影、过渡动画）
- `.podcast-3d-card-inner` - 卡片内部布局（渐变背景、居中显示图标和标题）
- `.podcast-3d-card.active` - 当前播放卡片的高亮样式（边框发光、标题变绿）
- `.podcast-3d-card-playing` - 播放指示器（脉冲动画）
- `.podcast-3d-card-glow` - 光晕效果
- 响应式：移动端自适应（768px以下缩小尺寸）

#### 2. 替换播放列表HTML
- 删除了原有的 `<div id="podcast-playlist">` 及其所有13个 `.podcast-track` 子元素
- 替换为3D轮播容器：
```html
<div id="carousel-stage-3d" class="carousel-stage-3d">
    <div class="carousel-track-3d" id="carousel-track-3d"></div>
</div>
```

#### 3. 添加3D轮播JavaScript逻辑
在 `</body>` 前添加了 `<script>` 代码块，包含：
- `Podcast3DCarousel` 类：
  - `constructor(containerId, items)` - 初始化容器和卡片
  - `render()` - 根据 `podcastList` 数据渲染卡片（显示图标和标题）
  - `bindEvents()` - 绑定鼠标和触摸事件（拖拽切换）
  - `handleStart/Move/End()` - 拖拽处理逻辑（阈值80px）
  - `updateStyles()` - 根据 `activeIndex` 更新所有卡片的3D变换和层级
  - `getCardStyle(index)` - 计算卡片的3D变换参数（translateX, scale, rotateY, translateZ, opacity）
  - `goTo(index)` - 切换到指定索引（循环），并调用 `playPodcast(index)`
  - `next()` / `prev()` - 下一首/上一首
  - `reRender()` - 重新渲染以更新播放指示器

- 初始化代码（不依赖DOMContentLoaded，因为<script>在页面底部）：
```javascript
if (window.podcastList && document.getElementById('carousel-track-3d')) {
    window.podcast3DCarousel = new Podcast3DCarousel('carousel-stage-3d', window.podcastList);
}
```

#### 4. 修改 playPodcast() 函数
在 `he-y-i.html` 的原有 `<script>` 标签内，修改 `playPodcast()` 函数，在设置 `currentPodcastIndex = index;` 后添加3D轮播同步代码：
```javascript
// 同步3D轮播
if (window.podcast3DCarousel && window.podcast3DCarousel.activeIndex !== index) {
    window.podcast3DCarousel.activeIndex = index;
    window.podcast3DCarousel.updateStyles();
    if (window.podcast3DCarousel.reRender) window.podcast3DCarousel.reRender();
}
```

由于 `prevPodcast()` 和 `nextPodcast()` 都会调用 `playPodcast()`，所以它们会自动同步3D轮播，无需额外修改。

#### 5. 清理冗余代码
- 删除了之前添加的wrapper `<script>` 代码块（冗余，因为已经直接修改了 `playPodcast()`）

### 技术要点
- **3D效果实现**：使用 CSS `perspective: 1200px` 和 `transform-style: preserve-3d` 创建3D视角
- **卡片位置计算**：`getCardStyle()` 根据与 `activeIndex` 的距离计算 `translateX`（±140px/级）、`scale`（递减）、`rotateY`（±12°/级）、`translateZ`（递减）、`opacity`（递减）
- **拖拽切换**：监听 `mousedown/mousemove/mouseup` 和 `touchstart/touchmove/touchend`，当拖拽距离超过80px时切换
- **流畅动画**：卡片变换使用 `transition: transform 0.5s cubic-bezier(0.175, 0.885, 0.32, 1.275)`（弹性效果）
- **播放指示器**：当前播放的卡片显示脉冲动画的 🎧 图标
